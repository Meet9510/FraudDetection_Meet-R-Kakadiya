Deep Dive
What is Sentinel AI?

Transparent, Explainable Machine Learning

Sentinel AI is a modern fraud intelligence console designed to replace outdated, rigid rule-based payment routing scripts. By employing machine learning classifiers, it dynamically evaluates the multi-dimensional risk index of incoming checkout requests.

Traditional systems look at simple parameters like transaction values or country listings. Sentinel AI evaluates complex relationships (e.g. issuer amount-to-mean ratios, domain risk factors, hourly checkout frequency metrics) in parallel.

01
Ingest Rails
Extracts transaction values, card issuing numbers, network brand codes, and billing domains.
02
Neural Gradient Classifier
The LightGBM model checks the scaled input array and outputs a probability score in <15ms.
03
Local Decision Attributions
SHAP waterfall calculations isolate exactly which features pushed the transaction score.
Value Proposition
How We Are Different
Legacy systems rely on rigid, manual rules that are slow and generate high false positives. Sentinel AI uses explainable machine learning to make instant, highly precise risk decisions.

Rigid Rules vs. Dynamic ML

Legacy filters block users based on static conditions (e.g. if amount > $1000). Sentinel AI evaluates over 30 transaction features simultaneously via a trained LightGBM gradient booster to calculate a custom risk index.

Explainable Decisions (SHAP)

No more guessing why a transaction was declined. Every Sentinel AI verdict is supplemented by a local SHAP explanation that breaks down exactly how much weight each factor contributed to the model score.

Sub-15ms Live Pipeline

Our ultra-lightweight inference pipeline operates with single-digit millisecond latency. Transactions are analyzed, explained, and logged in real-time without delaying payment processing paths.
